HTML (Hypertext Markup Language)

Historia de HTML hasta HTML5

HTML (Hypertext Markup Language) es el lenguaje fundamental utilizado para la creación de páginas web y ha experimentado una notable evolución a lo largo de las décadas.

Todo comenzó en 1993 cuando Tim Berners-Lee, el inventor de la World Wide Web, creó el primer lenguaje de marcado, HTML, que permitía la creación de documentos hipertextuales básicos. A medida que la web crecía, se hizo evidente la necesidad de expandir las capacidades de HTML.

En 1995, se publicó HTML 2.0, que incorporó mejoras en la estructura y la posibilidad de incluir formularios y tablas. Luego, HTML 3.2 (1997) introdujo el soporte para hojas de estilo CSS y tablas más avanzadas, marcando un paso importante hacia una presentación más sofisticada. Sin embargo, fue HTML 4.01, lanzado en 1999, el que brindó un mayor control sobre la presentación y una semántica más rica. A pesar de estas mejoras, aún carecía de elementos específicos para multimedia y gráficos. La evolución continuó con XHTML en 2000, que promovía una codificación más estricta y estructurada, fomentando la adopción de buenas prácticas de desarrollo web.

Finalmente, en 2014, HTML5 llegó al escenario y revolucionó la web. Esta versión trajo consigo una amplia gama de características innovadoras, como etiquetas semánticas (por ejemplo, <header>, <nav>, <section>, <article>), soporte integrado para audio y video, gráficos vectoriales con SVG y capacidades de almacenamiento local con Web Storage.

Usos de HTML5

  • Desarrollo de sitios web: HTML5 permite la creación de sitios web modernos con una estructura semántica, contenido multimedia enriquecido y una experiencia de usuario mejorada.
  • Aplicaciones web: HTML5 es esencial para el desarrollo de aplicaciones web que se ejecutan directamente en el navegador, eliminando la necesidad de descargar e instalar software adicional.
  • Juegos en línea: Gracias a tecnologías como Canvas y WebGL, HTML5 ha impulsado el desarrollo de juegos en línea, ofreciendo gráficos 2D y 3D impresionantes.
  • Reproducción de medios: La etiqueta <video> y <audio> permite la reproducción de medios sin necesidad de complementos como Flash.
  • Aplicaciones móviles: HTML5 se utiliza en el desarrollo de aplicaciones móviles multiplataforma, lo que facilita la creación de aplicaciones para múltiples sistemas operativos, como iOS y Android.
  • Realidad virtual y aumentada: HTML5 es fundamental en experiencias de realidad virtual y aumentada en la web, gracias a APIs como WebVR y WebXR.

CSS (Cascading Style Sheets)

Historia de CSS

Cascading Style Sheets (CSS) es un lenguaje de diseño web que se ha desarrollado a lo largo de las décadas para mejorar la presentación de las páginas web. CSS se introdujo por primera vez en 1996 como una forma de separar la estructura (HTML) y la presentación (estilo) de las páginas web. La especificación CSS1 se publicó en 1996, y desde entonces ha habido varias versiones, incluyendo CSS2 en 1998 y CSS2.1 en 2011.

CSS3 y sus características

CSS3, la versión más reciente del lenguaje de estilo en cascada, ha revolucionado la forma en que diseñamos y presentamos páginas web.

Entre sus innovadoras características se encuentran:

  • Selectores avanzados: CSS3 introduce selectores avanzados que permiten una mayor precisión en la selección de elementos en una página web. Estos selectores, como :nth-child y :not, proporcionan a los diseñadores la capacidad de aplicar estilos específicos a elementos concretos, lo que da lugar a diseños más personalizados y detallados.
  • Transiciones y animaciones: CSS3 ofrece la posibilidad de crear transiciones y animaciones fluidas en elementos HTML. Esto impulsa una experiencia web más interactiva al permitir que los elementos cambien de forma suave en respuesta a acciones del usuario, como pasar el ratón por encima o hacer clic en un botón. Las transiciones y animaciones son fundamentales para dar vida a las páginas web.
  • Sombras y bordes redondeados: Con CSS3, puedes aplicar sombras suaves y bordes redondeados a elementos en una página web. Estos efectos de estilo agregan un toque de diseño y profundidad a tus elementos, lo que hace que las páginas sean visualmente atractivas y agradables para los visitantes.
  • Fuentes web: CSS3 permite utilizar fuentes personalizadas en las páginas web, lo que mejora significativamente la tipografía y el aspecto general del texto. Esto otorga a los diseñadores un mayor control sobre la apariencia del contenido textual, lo que es esencial para la identidad y el estilo de una página web.
  • Media queries: CSS3 presenta las media queries, que posibilitan la creación de diseños responsivos. Esto significa que una página web puede adaptarse automáticamente a diferentes tamaños de pantalla y dispositivos, garantizando que el contenido se vea y funcione bien en cualquier contexto, ya sea en un teléfono móvil, una tableta o una computadora de escritorio.
  • Flexbox y Grid Layout: CSS3 introduce módulos como Flexbox y Grid Layout, que simplifican la creación de diseños complejos y flexibles en una página web. Estos sistemas de diseño proporcionan una mayor eficiencia en la organización de elementos en una página, lo que resulta en una mejor estructura y alineación de contenido.
  • Transformaciones 2D y 3D: CSS3 permite realizar transformaciones 2D y 3D en elementos, como rotación, escalado y traslación. Esto facilita la creación de efectos visuales llamativos y dinámicos, lo que es especialmente útil para resaltar elementos clave y mejorar la interacción del usuario en la web.
